💸 Spendzy

«Every Rupee. Every Insight.»

Spendzy is a modern personal finance platform that combines the power of an Android application with a beautiful web dashboard.

The Android app automatically detects supported bank transaction SMS messages (with user permission), categorizes expenses, and securely synchronizes them with the cloud. The web application provides powerful analytics, budgeting, reporting, and financial insights from the same data.

Vision

Create the smartest and most privacy-focused personal finance platform that helps users understand, control, and improve their spending habits.
